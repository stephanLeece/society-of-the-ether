# Society of the Ether.

This is a social network i made way back in 1867. ReactJS with Redux. Realtime chat using Socket.io.

## Technologies
- **Language**: Javascript
- **Frontend**: React, Redux
- **Backend**: Node, Express, Socket.io
- **Databases**: PostgreSQL, AWS S3
